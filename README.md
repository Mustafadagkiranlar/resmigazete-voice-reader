# Requirements

- Python 3.12.4

# Setup

Create a virtual environment for requirements in the root of the folder and install requirements.

```bash
python3 -m venv resmigazete
# activate venv
source resmigazete/bin/activate

python -m pip install -r requirements.txt
```
